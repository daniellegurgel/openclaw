# HEARTBEAT.md

## Check on every heartbeat:

1. Read `memory/YYYY-MM-DD.md` (today) — any scheduled callbacks or follow-ups due?
2. If a callback time has arrived → send the client a WhatsApp message resuming the conversation
3. If a follow-up was promised and info is now available → reach out to the client
4. If nothing is pending → reply HEARTBEAT_OK
